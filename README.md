# SimpleCommands
This plugin will help you to use commands faster and easier

<br>
## Commands
| **Command** | **Description** |
| --- | --- | --- |
| **/day** | **Time Set Day** |
| **/night** | **Time Set Night** |
| **/gms** | **Change Gamemode Survival** |
| **/gma** | **Change Gamemode Adventure** |
| **/gmc** | **Change Gamemode Creative** |
| **/gmspc** | **Change Gamemode Spector** |
| **/heal** | **Heal** |
| **/food** | **Food** |
